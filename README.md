# Learn React with TypeScript

This repository contains the example apps from the book [Learn React with TypeScript](https://www.packtpub.com/en-us/product/learn-react-with-typescript-9781836643166?srsltid=AfmBOoqeeSBi4BeAC8GpPZ4ilfTiuu9Xq7kanszxWt0HfMe3ziBD343J).

The `blog` directory contains the blog app, which uses Next.js and Tanstack Query

The `forms` directory contains the form app, which uses Next.js and React Hook Form
